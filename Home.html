UX/UI Design

UX design is different then UI design. UX or User eXperince design is a process of research, brain storming, implement, report and some time back to brainsstorming if the UI design or the project have been approved. UI design only take part in the implement section where the designer will design user interface and then UX will responsible on which place to start the road.

Design Fundamental

  • Color & Contrast :use similar but not too contrast

  • White Space :(Empty Space) keep a minimum space not too large or too small

  • Visual Hierarchy :Prioritize the grap-attention elements

  • Complexity VS Simplicity :don't over do just make it look like change but no change

  • Consistency :Don't mix font out just to look unique but need to feel right

  • Scale :Don't make thing too large or too small make thing feel right

The relation of these element have no other and for designer, they need a tester to test their DEMO in order to see from the user prospective and able to tell which place to change and what not.
